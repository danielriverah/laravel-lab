# Módulo de Filtros Dinámicos ERP

## Objetivo
Construir un motor de filtros dinámico configurable desde base de datos para controlar:

1. Filtros de datos
2. Filtros de catálogo

El backend debe estar preparado para frontend en Vue 2 dentro de Blade.

## Alcance de esta etapa
- Modelos Eloquent
- Repositorios
- Servicios
- CRUD backend
- Endpoints JSON
- Endpoint debug

## Fuera de alcance en esta etapa
- Componentes Vue 2
- Render visual del árbol
- Builder visual de filtros
- Rediseño de tablas
- Migraciones
