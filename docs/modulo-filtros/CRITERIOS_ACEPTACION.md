# Criterios de aceptación

## Estructura
- Existen modelos Eloquent para todas las tablas del módulo
- Existen relaciones correctas entre modelos
- Existen repositorios base del módulo
- Existen servicios principales del motor

## CRUD
- CRUD de módulos funcional
- CRUD de campos funcional
- CRUD de variables funcional
- CRUD de operadores funcional
- CRUD de filtros funcional
- CRUD de valores de filtro funcional
- CRUD de asignaciones a roles y usuarios funcional
- CRUD de overrides de usuario funcional

## Motor
- Puede aplicar filtros a Eloquent
- Puede aplicar filtros a Collection
- Puede aplicar filtros a array de objetos
- Resuelve variables dinámicas
- Combina valores estáticos y variables
- Aplica add/remove/replace

## Debug
- Existe endpoint debug
- El endpoint muestra filtros aplicados y valores resueltos

## API
- Todas las respuestas son JSON consistentes
- Los endpoints son consumibles por Vue 2
